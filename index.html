HTML

What is HTML

  • With HTML you can create your own Website
  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc

Parts in HTML

Tags

Tags can be defined as the instructions which are being directly embedded in the text of an HTML document. The types of tags used in the HTML document are responsible to tell a web browser to do something (follow the instruction) instead of just displaying text.

Types of tags in HTML

  • Paired Tags
  • Unpaired Tags

Paired Tags

  • An HTML tag is known as Paired tag when it contains an Opening as well as a Closing Tag.
  • Paired tag starts with an opening tag: the tag name enclosed inside the angle brackets, ex- and ends with an ending tag: the tag name starts with a forward slash & enclosed inside the angle brackets, ex- .

Unpaired Tags

  • An HTML tag is known as Empty tag when it contains only an Opening but no Closing Tag.
  • Unpaired tags are sometimes also named as Standalone Tags or Singular Tags since they do not require a companion tag.
  • Ex- hr, br, etc.
